# Fraud Detection & Customer Segmentation

## Project Overview
This project combines supervised and unsupervised machine learning techniques to detect fraudulent transactions and segment customers based on purchasing behavior. The objective is to improve risk management and support targeted marketing strategies.

## Objectives
- Detect fraudulent transactions using classification models
- Handle highly imbalanced datasets
- Compare multiple machine learning algorithms
- Segment customers using clustering techniques
- Generate actionable business insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- SMOTE
- Matplotlib
- Seaborn

## Project A: Fraud Detection

### Problem Statement
Identify fraudulent transactions from a highly imbalanced credit card transaction dataset.

### Techniques Used
- Data Cleaning & Preprocessing
- Feature Scaling
- SMOTE Oversampling
- Model Training & Evaluation
- Threshold Optimization

### Key Insights
- Improved fraud detection using class balancing techniques
- Compared model performance across multiple algorithms
- Optimized classification threshold to reduce financial risk

## Project B: Customer Segmentation

### Problem Statement
Group customers into meaningful segments to enable personalized marketing and business strategies.

### Clustering Techniques
- K-Means Clustering
- Hierarchical Clustering


### Key Insights
- Identified distinct customer groups
- Analyzed spending behavior and demographics
- Recommended targeted marketing strategies
